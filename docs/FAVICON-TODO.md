# Favicon Assets Needed

The following favicon files need to be created with the Moveroo brand logo:

- [ ] `favicon-16x16.png` - 16x16px PNG
- [ ] `favicon-32x32.png` - 32x32px PNG
- [ ] `apple-touch-icon.png` - 180x180px PNG for iOS devices

## Design Guidelines

- Use the Moveroo kangaroo logo
- Primary color: Emerald green (#10b981)
- Background: White or transparent
- Ensure the design is recognizable at small sizes

## Tools

You can use tools like:

- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Adobe Illustrator/Photoshop
- Figma

Once created, place these files in the `/public` directory and delete this file.
