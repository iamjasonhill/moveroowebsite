# Moveroo Homepage Audit

## Current Homepage Shape

The homepage in `src/pages/index.astro` is a custom branded page worth preserving.

Key sections currently present:

1. hero with the main Moveroo value proposition
2. three service/coordination tracks for household, vehicle, and added-complexity moves
3. explanation of the move-planning flow
4. feature, pricing, testimonials, and FAQ sections

## Messaging Anchors

- one calmer plan for complex moves
- household, vehicle, and specialty freight coordination
- conversion pathways into the quote system rather than on-site booking logic

## Things To Preserve

- the explicit service split between household, vehicle, and complexity/support work
- the strong route into external quoting
- branded OG/structured-data support already present on the homepage
