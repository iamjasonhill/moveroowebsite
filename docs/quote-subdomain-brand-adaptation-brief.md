# Moveroo Quote Subdomain Brand Adaptation Brief

## Scope

Apply the Moveroo brand to `quotes.moveroo.com.au` across:

- `/quote/household`
- `/quote/vehicle`
- `/booking/create`
- `/contact`
- `/customer/login`

## Primary Journey

`quotes.moveroo.com.au` is the canonical Moveroo-owned quote hub. Household quote is the primary journey, with vehicle quote, booking, contact, and customer portal retained as standard app pathways.

## Requirements

- Moveroo controls runtime forms, validation, analytics, route safety, and completion flow.
- This repo controls brand tokens, copy, visual direction, and canonical hostname.
- Preserve the existing linked GA4/runtime analytics contract for `quotes.moveroo.com.au`.
- Use the standard Moveroo navigation and relative app routes.
- Contact buttons must show only the shared phone and email buttons.
- Header and footer should use the Moveroo dark teal shell, aligned with the hero treatment.
- Active step, progress, selected choice, field focus, and validation states should use Moveroo emerald.
- Public quote surfaces should stay light-first; dark mode is not a separate public design requirement for this imported surface.

## Out Of Scope

Do not import or redesign `removalists.moveroo.com.au` in this handoff. It is a historical pilot surface until its current product role is confirmed.
